# catgloss

*a lightweight CSS framework for dreamy, cat-coded interfaces*

---

## Philosophy

catgloss isn't just a design system — it's a **mood**.

Inspired by VHS haze, iridescent shimmer, lavender tea, and the gentle chaos of cats.

**Design principles:**
- Soft over sharp
- Haze over clarity  
- Glow over shadow
- Whisper over shout

---

## Installation

### via unpkg (CDN)

```html
<link rel="stylesheet" href="https://unpkg.com/@shimmerwhere/catgloss@latest/dist/catgloss.css">
```

### via npm

```bash
npm install @shimmerwhere/catgloss
```

---

## Quick Start

```html
<!DOCTYPE html>
<html>
<head>
    <link rel="stylesheet" href="https://unpkg.com/@shimmerwhere/catgloss@latest/dist/catgloss.css">
</head>
<body class="bg-lavender-haze paw-8">
    <div class="velvet-edges soft-shadow paw-6 bg-rose-mist">
        <h1 class="text-neko">Hello, dreamy world</h1>
        <p class="text-static">Styled with catgloss</p>
    </div>
</body>
</html>
```

--static-gray: #B8B8C8;
--midnight-blur: #1A1A2E;

**Usage:**
```html
<div class="bg-lavender-haze text-neko">Dreamy content</div>
```

## Utilities

### Spacing (Paw-based system)

```html
<div class="paw-4">Padding: 16px</div>
<div class="m-paw-2">Margin: 8px</div>
<div class="whisker-gap">Padding: 48px (large gap)</div>
```

### Blur & Haze

```html
<div class="haze-light">Light blur backdrop</div>
<div class="haze-medium">Medium blur</div>
<div class="blur-dreamy">Dreamy blur filter</div>
```

### Softness (Rounded corners & shadows)

```html
<div class="soft-edges">8px border radius</div>
<div class="velvet-edges">16px border radius</div>
<div class="cloud-edges">24px border radius</div>

<div class="soft-shadow">Subtle shadow</div>
<div class="velvet-shadow">Medium shadow</div>
<div class="shimmer-glow">Iridescent glow</div>
```

## Why catgloss?

Because Tailwind is efficient, but it doesn't **dream**.

catgloss is for interfaces that feel like **memories you never had**.
